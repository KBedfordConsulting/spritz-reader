# Spritz Speed Reader

Speed read EPUBs and PDFs one word at a time using the Spritz RSVP technique. Runs entirely in your browser — no data leaves your device.

**[Try it live →]((https://kbedfordconsulting.github.io/spritz-reader/))**

## How It Works

Drop in an EPUB or PDF file and hit play. Words appear one at a time at a fixed focal point, with the center letter highlighted in red. Your eyes stay in one spot while words flow past — no scanning, no backtracking. Most people comfortably read 2-3x faster this way.

## Features

- **100–900 WPM** adjustable speed with smart punctuation pausing
- **Chapter navigation** via Table of Contents sidebar
- **Skip controls** — jump forward/back by 10 or 30 seconds
- **Keyboard shortcuts** — Space (play/pause), arrows (skip/speed), T (TOC)
- **Progress tracking** — clickable progress bar, time remaining, chapter labels
- **Fully private** — all parsing happens locally, nothing is uploaded anywhere

## Supported Formats

- **EPUB** — standard and single-file layouts, EPUB2 and EPUB3
- **PDF** — with outline-based chapter navigation when available

## Usage

Open `index.html` in any modern browser. That's it — no install, no server, no build step.

## Keyboard Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play / Pause |
| `←` / `→` | Skip back / forward 10s |
| `Shift+←` / `Shift+→` | Skip back / forward 30s |
| `↑` / `↓` | Speed up / slow down |
| `T` | Open Table of Contents |
| `Escape` | Close TOC |
